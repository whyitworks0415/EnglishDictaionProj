# EnglishDictationProj

## 실행 방법
1. index.html실행
2. 형식에 맞는 파일 업로드
3. 스크립트 보기로 지문을 볼 수 있음
	- 각각의 지문을 누르면 그 지문의 mp3음원을 들을 수 있음
	- 지문의 단어를 누르면 그 단어가 하이라이트 됨
	- 단어 하이라이트를 누르면 vocab.xlsx파일에 있는 단어가 하이라이트됨
4. 모의 테스트로 테스트를 할 수 있음
	- hax는 1개에서 3개의 빈칸 묶음의 갯수를 말함(최대 갯수는 문장의 갯수와 동일)
	- hax한개의 빈칸의 갯수가 그 hax의 단어 갯수임
	- 처음 빈칸에 단어를 적고 space를 누르면 그 다음 단어로 넘어가짐(다음 hax로는 넘어가지지 않음(tab과 마우스를 통하여 전환))
	- 총 2번 들을 수 있지만 위의 플레이 바를 통하여 정지 및 다시 들을 수 있음
	- 체점을 통하여 체점할 수 있음(대문자, 쉼표, 마침표 고려하지 않음, 따옴표는 고려함)
	- 틀린 답은 그 답을 누르거나 커서를 대면 실제 답이 보임
5. 다크 모드를 통하여 테마를 블랙, 화이트로 전환할 수 있음
